backend and it's types - backend არის ის ნაწილი რომელსაც მომხმარებელი ვერ ხედავს ვებსაიტზე შესვლისას. ის შედგება ორი ნაწილისგან სერვერისგან და მონაცემთა ბაზისგან.
request, response - როდესაც გვინდა ინფორმაცია გადავაგზავნოთ მონაცემთა ბაზაში, მაგ დროს სერვერზე ვვგზავნით request-ს ანუ მოთხოვნა თუ ის დადებითია მაშინ გადადის მონაცემთა ბაზაში და იქიდან ჯერ სერვერზე ხოლო შემდეგ front-end ზე გადადის response ანუ შეინახა თუ არა მონაცემთა ბაზამ.
database and it's types - database არის ინფორმაცია რომელსაც ინახავს front-end-იდან. ის ორი სახისაა NoSQL და SQL. SQL არის ცხრილის სახით და NoSQL არის ან key-value სახით ან დოკუმენტებად.
connection among frontend, server and database - როდესაც გვინდა ინფორმაცია გადავაგზავნოთ მონაცემთა ბაზაში, მაგ დროს სერვერზე ვვგზავნით request-ს ანუ მოთხოვნა თუ ის დადებითია მაშინ გადადის მონაცემთა ბაზაში და იქიდან ჯერ სერვერზე ხოლო შემდეგ front-end ზე გადადის response ანუ შეინახა თუ არა მონაცემთა ბაზამ.
