3) თქვენი სიტყვებით ახსენით url რამდენი ნაწილისგან შედგება და განმარტეთ თითოეული ნაწილი

url არის უნიკალური მისამართი რომელიც ყველა ვებსაუტს გააჩნია. იგი შედგება 4 ნაწილისგან: protocol, domain, path და query

1. protocol - იგივე https ან http. იგი ყველა ვებსაიტს სჭირდება და არ არის optional.
2. domain - განსაზღვრავს თუ რომელ ვებსაიტზე შევიდეს და არ არის optional.
3. path - განსაზღვრავს ვებსაიტის რომელ გვერდში შევიდეს და არის optional.
4. query - ირჩევს ამ გვერდის კონკრეტულ ელემენტს და არის optional.
